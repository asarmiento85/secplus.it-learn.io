The 5 Exam Domains

  • Domain 1.0 — General Security Concepts (12%) — Security controls, CIA triad, cryptography, change management
  • Domain 2.0 — Threats, Vulnerabilities, and Mitigations (22%) — Threat actors, attack vectors, vulnerability types, malware, mitigation

  • Domain 4.0 — Security Operations (28%) — Hardening, IAM, vulnerability management, monitoring, incident response
  • Domain 5.0 — Security Program Management and Oversight (20%) — Governance, risk management, compliance, audits, awareness
